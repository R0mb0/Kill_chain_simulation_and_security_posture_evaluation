Phase 3.2: Web Enumeration via DIRB

Objective: Map the directory structure of the target web server (Port 80) to uncover hidden paths, administrative panels, and vulnerable applications (e.g., DVWA, Mutillidae).

1. Methodology

Following the discovery of the Apache web server, a directory busting tool (DIRB) was utilized to brute-force web paths using a predefined wordlist.

The following command was executed on the Kali Linux machine:

dirb http://192.168.100.20/

2. Expected Directories of Interest

We are specifically looking for the following endpoints to proceed with the Web Application Exploitation phase:

  • /dvwa/ - Damn Vulnerable Web App (Primary target)
  • /mutillidae/ - OWASP Mutillidae II
  • /phpMyAdmin/ - Database administration interface
